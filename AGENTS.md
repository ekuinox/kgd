# エージェント向け指示

作業前に [CONTRIBUTING.md](CONTRIBUTING.md) を確認すること。

このリポジトリでは、次の事項を守ること。

- 開発ルールと Rust のスタイル規約は `CONTRIBUTING.md` に従うこと。
- 変更完了前に `just validate` を実行すること。
- 依存関係の追加・更新は `CONTRIBUTING.md` の workspace ルールに合わせること。
- コーディングエージェントを使って作成または更新したコミットには、必要に応じて `Co-authored-by: Codex <codex@openai.com>` 形式のトレーラーを付与すること。

矛盾がある場合は `CONTRIBUTING.md` を優先すること。
